TapLocal
A full-stack marketplace platform that connects homeowners and individuals with trusted local service providers — from handymen to cleaners to general contractors.
Features

Service Provider Listings & Search — Browse and filter local professionals by service type and location
Booking & Scheduling System — Request and confirm service appointments directly through the platform
User Authentication — Secure signup/login for both customers and service providers

Tech Stack
Frontend

React.js + TypeScript
Vite
TailwindCSS

Backend

Node.js + Express.js
RESTful API architecture
JWT-based authentication

Getting Started
bash# Clone the repo
git clone https://github.com/demilade111/taplocal.git
cd taplocal

# Install dependencies
npm install

# Start the development server
npm run dev

About
TapLocal was built to solve the friction of finding and booking reliable local service professionals. Instead of relying on word-of-mouth or scattered listings, users can discover, vet, and book providers in one place.
